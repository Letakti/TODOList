# TODOList

Небольшое JavaFX‑приложение для ведения списка задач с приоритетами, темами оформления и автосохранением.

## Быстрый старт

**Требования:**
- JDK 17
- Maven (или запуск из IDE)

**Сборка и запуск через Maven:**
```bash
mvn clean package
```

Запуск через IDE: открой проект и запусти `org.letakti.todolist.Launcher`.

## Функции

- Добавление задач
- Удаление задач
- Отметка “Выполнено”
- Приоритеты задач (Высокий/Обычный/Низкий)
- Сортировка по приоритету и времени обновления
- Темы и акценты
- Автосохранение

## Управление

**Горячие клавиши:**
- `F2` — редактировать выбранную задачу
- `Delete` — удалить выбранную задачу
- `Ctrl+1` — высокий приоритет
- `Ctrl+2` — обычный приоритет
- `Ctrl+3` — низкий приоритет

**Меню и контекстное меню:**
- Меню “Редактировать” содержит редактирование и приоритеты
- ПКМ по задаче открывает контекстное меню с теми же действиями

## Хранение данных

Задачи сохраняются в файле:
- `user.home/todolist_saved.json`

Формат хранит поля:
- `text`
- `done`
- `createdAt`
- `updatedAt`
- `priority`

При первом запуске возможна миграция со старого `todolist_saved.txt` (если он есть).

## Темы и акценты

В меню “Настройки” можно выбрать:
- Тема: светлая / тёмная / как в системе
- Акцент: мята / синий / оранжевый

Настройки сохраняются в:
- `user.home/todolist_settings.json`

## Структура проекта (ключевые файлы)

- `src/main/java/org/letakti/todolist/TODOListController.java` — логика UI, список задач, меню
- `src/main/java/org/letakti/todolist/TaskStorage.java` — загрузка/сохранение задач
- `src/main/resources/org/letakti/todolist/style.css` — стили интерфейса
